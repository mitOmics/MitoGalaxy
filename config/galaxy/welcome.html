Welcome to Galaxy for Genome Annotation!


To get started, upload a fasta genome, and some gff3 formatted datasets. The JBrowse tool will let you build up a JBrowse instance which is viewable inside of Galaxy.

Once you've done that, you can Create or Update an organism in Apollo with that JBrowse instance.
